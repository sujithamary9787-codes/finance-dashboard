# Finance Dashboard

## Overview
This is a simple finance dashboard built using Streamlit.
It allows users to view financial summary, transactions, charts and insights.

## Features
- Dashboard summary cards
- Balance trend chart
- Spending breakdown chart
- Transactions table
- Search and filtering
- Role based UI (Viewer / Admin)
- Add transaction (Admin)
- Insights section

## Tech Stack
- Python
- Streamlit
- Pandas
- Plotly

## How to Run
pip install streamlit pandas plotly
streamlit run app.py****
